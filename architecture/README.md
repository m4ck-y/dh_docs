# Especificaciones de Arquitectura

El objetivo de esta carpeta es almacenar la documentación técnica detallada sobre la estructura y el comportamiento del sistema.

## Contenido
- **[Diagrama de Arquitectura Global](diagrama_arquitectura.md)**: Visión de alto nivel del sistema.
- **Diagramas de Secuencia**: Interacción entre microservicios.
- **Modelos de Datos (ERDs)**: Estructura de la base de datos centralizada.
- **Definición de APIs**: Contratos y esquemas de datos compartidos.

Estos documentos son esenciales para asegurar que los nuevos microservicios sigan los mismos patrones que los existentes.
