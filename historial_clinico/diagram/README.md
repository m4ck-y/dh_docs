# Clinical History Workflow Diagrams (Mermaid)

Source for machine-readable clinical workflows migrated from master Draw.io file.

## Core Workflows
- [A_REGISTRO.mmd](./A_REGISTRO.mmd): Patient demographics and registration intake.
- [B_ANAMNESIS.mmd](./B_ANAMNESIS.mmd): Initial reason for consultation and clinical triage.
- [C1_PADECIMIENTO_ACTUAL.mmd](./C1_PADECIMIENTO_ACTUAL.mmd): Detailed current illness symptom tracking.
- [C2_APP.mmd](./C2_APP.mmd): Personal Pathological History (chronic diseases, surgeries).
- [D_AHF.mmd](./D_AHF.mmd): Family Medical History (Heredo-Familiares).
- [E_HISTORIA_DESARROLLO.mmd](./E_HISTORIA_DESARROLLO.mmd): Pediatric growth milestones and psychomotor development.
- [F_PROGRAMAS_PREVENTIVOS.mmd](./F_PROGRAMAS_PREVENTIVOS.mmd): Vaccination schedules and preventive screenings.
- [G_SALUD_SEXUAL_REPRODUCTIVA.mmd](./G_SALUD_SEXUAL_REPRODUCTIVA.mmd): Family planning and sexual health.
- [H_EMBARAZO.mmd](./H_EMBARAZO.mmd): Pregnancy history and current status tracking.
- [I_PRENATAL.mmd](./I_PRENATAL.mmd): High-fidelity prenatal control and clinical services.

## System Logic & Annexes
- [ANNEX_A_VIAS_ADMINISTRACION.mmd](./ANNEX_A_VIAS_ADMINISTRACION.mmd): Catalog of pharmaceutical forms and administration routes.
- [ANNEX_B_CONDITIONAL_LOGIC.mmd](./ANNEX_B_CONDITIONAL_LOGIC.mmd): Data dependency map (Age/Sex triggering sections).
- [ANNEX_C_FORM_CONNECTIONS.mmd](./ANNEX_C_FORM_CONNECTIONS.mmd): Specialized questionnaire triggers (ZARIT, SF-12, ASQ-15, HADS).

## Usage
These `.mmd` files are the Technical Source of Truth for frontend form logic and visibility rules. Use a Mermaid renderer or VS Code extension to visualize.
