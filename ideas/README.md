# Ideas y Propuestas (RFCs)

El objetivo de esta carpeta es servir como un espacio de "brainstorming" y diseño antes de convertir una idea en una tarea real.

## Flujo Sugerido
1. **Creación**: Escribe una nueva idea en un archivo `.md`.
2. **Discusión**: Refinamos la idea juntos (Human + IA).
3. **Conversión**: Una vez que la idea está madura, se desglosa en tareas y se mueve al `backlog.md`.

Este proceso evita saturar el backlog con tareas que aún no están bien definidas técnicamente.
