# nom024

Documentación de referencia sobre la NOM-024-SSA3-2012 y sus estándares de interoperabilidad en salud.

La NOM-024 establece los criterios de intercambio de información clínica en México, incluyendo el diccionario de datos del Subsistema de Prestación de Servicios (SIS) — Consulta Externa.
