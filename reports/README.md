# Reportes y Auditoría

El objetivo de esta carpeta es almacenar resultados de análisis automáticos y manuales.

## Ejemplos de Reportes
- **Seguridad**: Escaneos de vulnerabilidades.
- **Rendimiento**: Resultados de pruebas de carga.
- **Calidad de Código**: Reportes de cobertura de tests o análisis estático.

Mantener estos reportes aquí permite trazar la evolución de la salud del código a lo largo del tiempo.
