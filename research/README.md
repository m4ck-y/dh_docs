# Investigación y Hallazgos (Research)

Esta carpeta contiene investigaciones, comparativas técnicas y análisis detallados que preceden a las decisiones de arquitectura.

## Objetivo
El objetivo es documentar el proceso de exploración técnica para que el equipo entienda qué opciones se evaluaron y bajo qué premisas se realizaron los hallazgos.

## Flujo de Trabajo
1. **Identificar Necesidad**: ¿Faltan datos para tomar una decisión?
2. **Investigar**: Crear un documento en esta carpeta con los hallazgos.
3. **Proponer Decisión**: Basándose en el research, proponer una entrada en `docs/decisions/`.
