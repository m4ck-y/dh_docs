## 🛠️ Cómo crear una nueva tarea
Para iniciar una nueva tarea, copia la carpeta **[`_TEMPLATE/`](_TEMPLATE/README.md)** y asígnale el siguiente número disponible (`TASK-NNN`).

### Estructura de una Tarea:
- `planning/`: Plan de ejecución y análisis técnico.
- `progress/`: Registros de sesión diarios.
- `artifacts/`: Scripts, diagramas y otros archivos temporales.
- `README.md`: Resumen y estado de la tarea.
