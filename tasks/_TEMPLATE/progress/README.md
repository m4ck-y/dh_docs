# Registro de Progreso

Esta carpeta contiene los logs diarios o por sesión de la tarea. Ayuda a mantener una trazabilidad clara de qué se hizo, qué decisiones se tomaron en el momento y qué bloqueadores aparecieron.

## Formato Sugerido
Los archivos deben nombrarse siguiendo el patrón `YYYY-MM-DD.md`.

## Ejemplo de Contenido
- Resumen de la sesión
- Tareas completadas
- Decisiones técnicas tomadas durante la ejecución
- Siguientes pasos o bloqueadores
